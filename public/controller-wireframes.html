Lely Horizon — UX/UI Targeting & Layout Iteraties

Wanneer een speler een interactieve spell speelt zoals Counterspell, moet de controller transformeren om de benodigde targeting-input te vangen zonder de algehele layout te breken. Hieronder staan drie concrete architectuur-opties voor Codex.
Optie 1

Contextual Overlay

De 3-koloms basis blijft muurvast staan voor je spiergeheugen. Zodra targeting nodig is, zakt de handwaaier volledig weg en maakt plaats voor de actieve Stack.

  • UX-Gevoel: Alsof je een fysieke controller vasthoudt. Knoppen verschuiven nooit.
  • Mechaniek: `HandFan` animeert naar `y: 400` via Framer Motion. Tegelijkertijd fadet de `StackView` in op dezelfde plek.
  • Codex Focus: Geen zware herstructurering van grid kolommen nodig. Vaste layout-omgeving.
// Codex State Switch:
isTargeting ? <StackOverlay /> : <HandFan />
Optie 2

Modal Focus

Maximale rust en nul foutmarges. Zodra er een target vereist is, schuift er een full-screen modal over de controller heen om de invoer af te dwingen.

[STACK ITEM #1] P2: Lightning Bolt
CONFIRM CAST
ANNULEREN
  • UX-Gevoel: Absoluut gefocust. Het sluit fouten uit omdat de speler niks anders kan indrukken.
  • Mechaniek: Een absolute overlay div `fixed inset-0 z-[200]` die over de complete JSX heen rendert op basis van `isTargeting === true`.
  • Codex Focus: Veiligste optie voor Codex. Geen enkele kans op layout-shifting in je hoofdgrid.
{isTargeting && <FullScreenTargetingModal />}
Optie 3

Picture-in-Picture Tab

Gebruikt het tab-systeem dat je al hebt (Hand / Board). Er opent automatisch een tijdelijke, rode "Stack" tab in het centrum van je huidige grid.

  • UX-Gevoel: Systeemgedreven en overzichtelijk. Maakt herkenbaar gebruik van de bestaande navigatiestructuur.
  • Mechaniek: De `centerViewMode` state springt automatisch van `'hand'` naar `'stack'` zodra `requiresTarget` activeert.
  • Codex Focus: Zeer makkelijk te integreren in je huidige state, omdat je de JSX structuur van het midden-panel hergebruikt.
setViewMode('stack'); // Getriggerd door spell drop
