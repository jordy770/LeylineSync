Keyword-icons — opties uit game-icons

Per effect een paar geldige game-icons naast elkaar, zoals ze op een kaart zouden staan (gekleurd chipje) + groter ter referentie. ★ = mijn huidige keuze. Laat weten welke je per effect wilt.

Klein chipje = weergave op de creature-tegel · grote glyph = het kale icoon

Allemaal echte game-icons (hier via Iconify gerenderd, voor productie installeren we de set lokaal — CC-BY 3.0, naamsvermelding volstaat).
